Shift+I` (Windows) / `Cmd+Opt+I` (Mac): Open DevTools
 - `Ctrl+Shift+C`: Select element to inspect
 - `Ctrl+L`: Clear console
 - `$0`: Reference currently selected element
 - `$$('selector')`: Query selector all (returns array)
